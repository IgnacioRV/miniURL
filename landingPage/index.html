URL Shortener

Created by Ignacio Rasche for Free Code Camp

Example of usage:

Create a new short link:

https://minifyer.herokuapp.com/new/https://WWW.EXAMPLE.COM

In the response page you will get a JSON document with the link you provided and the short link, for example:

{"old":"https://docs.mongodb.com","new":"https://minifyer.herokuapp.com/27371"}

You can also use this handy button below :)
